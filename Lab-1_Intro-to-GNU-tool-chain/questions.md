1. What is "free" or Libre software?
Free/Libre software is software free to use, distribute, change and study.
2. Is "free" software free.
Free software has nothing to do with it's price, is about the freedom of use, change, distribute and study the software.
3. What does the "ls" command do?
Lists the contents of the directory it is ran from.
4. What is an absolute path?
The absolute path is the path from the root directory, accesed with the ~ symbol(for example ~/dev)
5. What is a relative path?
A relative path is a path that is relative to the location/path where the shell is open in.
